---
title: Installation
---

Unity Editor 2019.4.9f1 or greater is required.

Download [`.unitypackage`](https://github.com/Web3Auth/web3auth-unity-sdk/releases/latest) from our latest release and import the package file into
your existing Unity3D project.

:::warning

You may encounter errors when importing this package into your existing project.

`The type or namespace name 'Newtonsoft' could not be found (are you missing a using directive or an assembly reference?)`

To fix this problem you need to add the following line into dependencies object which is inside the `Packages/manifest.json` file.

```json
"com.unity.nuget.newtonsoft-json": "3.0.2"
```

:::
